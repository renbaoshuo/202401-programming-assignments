# 6-4 其右上三角（含主对角线）元素之和。

输入二维数组的所有元素，求二维数组右上三角（包括主对角线）元素之和。

### 函数接口定义：
```c
在这里描述函数接口。例如：
int fun(int a[3][3]);
```
### 裁判测试程序样例：
```c
在这里给出函数被调用进行测试的例子。例如：
#include <stdio.h>

int fun(int a[3][3]);

int main()
{   int i,j,s,x[3][3];;
   for(i=0;i<3;i++)
      for(j=0;j<3;j++)
        scanf("%d",&x[i][j]);
   s=fun(x);
   printf("sum=%d\n",s);
   return 0;
}
/* 您的答案将被嵌入在这里 */
```

### 输入样例：
```in
1 2 3 4 5 6 7 8 9
```

### 输出样例：
```out
在这里填写相应的输出
sum=26
```
