# 7-1 寻找自守数

所谓自守数(也称守形数)，是指其平方数的低位部分恰为该数本身的自然数。例如：$$25^2 = 625$$，因此25是自守数。（注：0 和 1 也算自守数。）

请编写程序，输出指定范围内的所有自守数。若指定范围内不存在自守数，则输出None。  

#### 输入格式

正整数 $$a$$ 和 $$b$$, 且 $$a \leq b \leq 10000$$

#### 输出格式

若 $$[a, b]$$ 内存在自守数，则按由小到大的顺序输出，每行输出一个自守数；若不存在自守数，则输出None。

#### 输入样例1
```in
10 80

```
#### 输出样例1
```out
25
76

```
#### 输入样例2
```in
400 600

```
#### 输出样例2
```out
None

```
